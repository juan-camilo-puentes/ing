Objetivos específicos

•Identificar las necesidades del cliente mediante un análisis del manejo de los empleados y una entrevista

•Definir los roles y ramas de cada integrante de la empresa para poder adecuarlos a la base de datos y así dividir el ingreso de empleado – gerente a la aplicación

•Definir las herramientas y requerimientos para obtener el conocimiento para la realización de la aplicación
