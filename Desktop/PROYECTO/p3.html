Explicación del proyecto

•El proyecto va enfocado hacia el desarrollo de una aplicación y una pagina web en la cual en dicha aplicación, los moderadores u empleados (encargados) de la empresa event plus s.a.s tengan un seguimiento claro de los elementos que requieran utilizar para un evento mediante una base de datos en la cual tengan acceso a los diferentes cambios y variables de precios, inversión y perdidas que podrían obtener por cada evento que realice la empresa, teniendo en cuenta los cambios en la sociedad si se eleva el precio del producto o disminuye el precio del producto y el seguimiento de cada elemento del que se disponga a conseguir para el evento dependiendo el cliente y así mismo en dicha aplicación incluir un diagrama en el que permita dar seguimiento a cada resultado de cada evento en cuestión de gastos

•Se busca el diseño de una pagina web para dar una mejor visualización de el manejo de la empresa los distintos servicios que brinda y la dinámica organizacional para cada petición del cliente y información de la empresa, así mismo ofrecer una nueva publicidad para la empresa y llegar a diferentes tipos de clientes como eventos campestres o rurales.
